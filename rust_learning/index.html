Rust Template System

Rust 学习模板首页

这个目录不是数学课程内容，而是把 `math_univers` 的设计风格抽象出来，改造成 Rust 学习内容可复用的模板系统。

🎯 当前模板要解决的问题

  • 统一 Rust 学习页面的结构与阅读节奏
  • 把设计风格和主题内容彻底分离
  • 为后续子 agent 生成页面提供稳定骨架
  • 把第一步的风格学习结果固定下来，避免再次跑偏

1. 已学习到的设计特点

🧭

迁移模板时，真正需要学习的是“组织方式”和“阅读体验”，不是把原主题照搬进 Rust 仓库。

  • 深色阅读界面 + 高亮强调色
  • 侧边栏导航 + 单元分组
  • 卡片化知识块 + 统一标题层级
  • 快速测试、折叠答案、提示框、代码展示
  • 围绕“为什么 / 怎么用 / 会错在哪”来组织内容

## 第二步：章节规划
### Ch 1.2 Borrowing
- 目标：让读者把 Borrowing 看成 Ownership 的延伸，而不是一组零散限制。
- 知识点规划：
  1. 借用存在的原因与不可变借用的只读语义
  2. 可变借用的独占规则
  3. 借用作用域与最后一次使用
  4. 函数参数中的引用设计（`&str`、`&T`、`&mut T`）
  5. 借用检查器心智模型与同时可变/不可变借用的常见错误
- 快速测试方向：围绕“是否 move、何时可变、借用何时结束、签名怎么选、如何修复 E0502 类错误”展开。
- 练习梯度：基础签名选择题 → 可变借用练习 → 报错改写题 → API 设计题。

### Ch 1.3 Lifetimes
- 预规划方向：引用为什么需要生命周期标注、生命周期与借用规则的边界、常见函数签名案例。

### Ch 1.4 所有权的实际影响
- 预规划方向：所有权系统对 API 设计、集合操作、错误处理和工程代码结构的影响。
